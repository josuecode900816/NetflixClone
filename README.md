
# Ionic v4 Tutorial: NetflixClone 
Una aplicación demo NetflixClone.

# ¿Cómo ejecutarlo localmente?
Clona este repositorio con el siguiente comando: git clone https://github.com/josuecode900816/NetflixClone.git

Abre un terminal del sistema operativo y navega al repositorio en la carpeta

Ejecuta el comando "npm install" para instalar las dependencias requeridas del proyecto

Ejecuta el comando "ionic serve" para observalo en el browser localmente

Para simular un teléfono, si se ejecuta en el sistema operativo Windows presiones F12.
